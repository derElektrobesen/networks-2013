2. Протокол обмена
client:    error\0pack-id\0piece_num\0\hash\0file_name

server:    error\0pack-id\0piece_num\0piece_len\0\hash\0file_name\0data
    
    pack-id:Ключ, однозначно идентифицирующий клиентов (num) (2 bytes)
    len:    Длина data (4 bytes)
    action: 3 bytes
        byte 1: data bits
            0:  hash-bit
            1:  file-number
            2:  piece-number
            3:  data
            4:  file-path
        byte 2: action bits
            0:  search-file
            1:  download
            2:  shutdown-server /* ? */
        byte 3: status bits
        byte 4: error bits
            0:  file-not-found
            1:  piece-not-found
            2:  other-error

6. Возможные события клиента

        Передается имя файла
        Передается хэш-сумма
        Сравнение всех хэш-сумм
        Присвоение файлу номера заданного сервером
